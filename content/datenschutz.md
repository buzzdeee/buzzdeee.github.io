## 2. Hosting über GitHub Pages
Diese Website wird über GitHub Pages bereitgestellt, ein Dienst der GitHub Inc., 88 Colin P Kelly Jr St, San Francisco, CA 94107, USA.

Beim Aufruf der Website werden durch GitHub automatisch Informationen erfasst und in Server-Logfiles gespeichert. Dies sind insbesondere:
* IP-Adresse
* Datum und Uhrzeit der Anfrage
* Browsertyp und Browserversion
* Betriebssystem
* Referrer-URL

Die Verarbeitung erfolgt zur Sicherstellung eines störungsfreien Betriebs der Website sowie zur Verbesserung des Angebots. Die Datenverarbeitung erfolgt auf Grundlage von Art. 6 Abs. 1 lit. f DSGVO (berechtigtes Interesse).

Es kann nicht ausgeschlossen werden, dass Daten in die USA übertragen werden. GitHub ist unter dem EU-U.S. Data Privacy Framework zertifiziert.

Weitere Informationen findest du in der Datenschutzerklärung von GitHub: [GitHub Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement)

## 4. Webanalyse mit GoatCounter
Diese Website nutzt GoatCounter, einen datenschutzfreundlichen Webanalysedienst.

GoatCounter verwendet keine Cookies. Es werden lediglich anonymisierte Zugriffsdaten erhoben, um die Nutzung der Website statistisch auszuwerten. Erfasste Daten können sein:
* gekürzte/anonymisierte IP-Adresse
* aufgerufene Seiten
* Referrer
* Zeitpunkt des Zugriffs

Die Verarbeitung erfolgt auf Grundlage von Art. 6 Abs. 1 lit. f DSGVO.

Weitere Informationen: [GoatCounter Privacy](https://www.goatcounter.com/privacy)

## 5. Externe Links, Affiliate-Links, Social Media und YouTube
Diese Website enthält Links zu externen Websites (z. B. Amazon, eBay, Prusa oder andere Anbieter) sowie zu Social Media Profilen (z. B. Instagram, Twitter, LinkedIn).

Wenn du auf einen solchen Link klickst, wirst du auf die jeweilige externe Website weitergeleitet. Dort gelten die Datenschutzbestimmungen des jeweiligen Anbieters.

Einige externe Links können sogenannte Affiliate-Links sein. Wenn du über einen solchen Link einkaufst, erhalten wir ggf. eine Provision. Für dich entstehen dadurch keine zusätzlichen Kosten.

Beim Anklicken von Social Media Links werden ebenfalls Daten an die Betreiber der Plattformen übertragen. Bitte beachte die Datenschutzbestimmungen der jeweiligen Anbieter.

Diese Website kann YouTube-Videos enthalten, die über den Hugo YouTube Shortcode eingebettet sind. Der Blog hat die „Privacy Enhanced“-Funktion aktiviert, deshalb werden von YouTube erst dann Daten wie IP-Adresse und Browserinformationen verarbeitet, wenn das Video abgespielt wird. Weitere Informationen findest du in der Datenschutzerklärung von Google: [Google Datenschutzerklärung](https://policies.google.com/privacy)
