## 2. Hosting via GitHub Pages
This website is hosted via GitHub Pages, a service provided by GitHub Inc., 88 Colin P Kelly Jr St, San Francisco, CA 94107, USA.

When you visit this website, GitHub automatically collects information and stores it in server log files. This includes, in particular:
* IP address
* Date and time of the request
* Browser type and browser version
* Operating system
* Referrer URL

The processing is carried out to ensure the trouble-free operation of the website and to improve the service. The data processing is based on Art. 6 Para. 1 lit. f GDPR (legitimate interest).

It cannot be excluded that data will be transferred to the USA. GitHub is certified under the EU-U.S. Data Privacy Framework.

For more information, please see GitHub's privacy policy: [GitHub Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement)

## 4. Web Analytics with GoatCounter
This website uses GoatCounter, a privacy-friendly web analytics service.

GoatCounter does not use cookies. Only anonymized access data is collected to statistically evaluate the use of the website. Collected data may include:
* Shortened/anonymized IP address
* Pages visited
* Referrer
* Time of access

Processing is based on Art. 6 Para. 1 lit. f GDPR.
